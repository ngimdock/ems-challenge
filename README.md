# Employee Management System

A simple **Employee Management System** built with **React.js**, **SQLite**, and **React Router**, allowing HR personnel to manage employees and timesheets efficiently.

---

## Style

The app uses the [**shadcn/ui**](https://ui.shadcn.com/blocks) library for simple, reusable components. Minimal custom styling is applied, focusing on usability and a clean interface.

---

## Functionality

### Employees

- Create a new employee
- Update existing employee information
- Display a list of all employees
- View details of a specific employee
- Search employees by name, email or job title
- Pagination for employee lists

### Timesheets

- Create a new timesheet
- Update existing timesheets
- Display a list of timesheets in **calendar view**
- Display a list of timesheets in **table view**
- Toggle between calendar and table view
- View timesheets associated with a specific employee
- Pagination for timesheet lists

### Bonus

- Small dashboard

---

## Run the App

To set up and run the application locally:

```bash
npm install
npm run setup_db   # Create the database
npm run seed       # Seed the database with sample data
npm run build      # Build the application
npm run start      # Start the server
```
